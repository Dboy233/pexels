# pexels flutter lean demo flutter学习demo

Pexels 的API访问需要翻墙。

Pexels 官网:[https://www.pexels.com/zh-tw/](https://www.pexels.com/zh-tw/)

自己注册一个账号: 将Auth的认证修改为自己的。

目前完成的功能：

- 首页：精选图片推荐
- 收藏功能：只能查看在网页端的收藏，API没有提供收藏功能。
- 下载：后期增加。
- 搜索：搜索图片，仅限英语，毕竟国外API。
